# Dashboard TV Mode
Full-screen ambient dashboard for [[CasaTAO Studio]] and [[ARC Anandaz]] wall displays.
Shows: [[CasaTAO Energy System]] live stats, [[Frigate NVR]] camera feeds, [[Y-WORLD Dashboard]] project status.
Built with React, auto-refreshes every 30s via [[Supabase DB]] real-time.
