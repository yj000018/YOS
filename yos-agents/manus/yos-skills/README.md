# yOS Skills

Manus agent skills for the Y-OS cognitive operating system.

## Skills

| Skill | Description |
|---|---|
| `memoriser` | Generate yOS Memory Packages from any content and push to Notion Inbox |
| `hydrater` | Retrieve memory context from Notion and generate Context Injection Packages |
| `memory-manager` | Persistent memory system using Notion |
| `session-synthesis` | Generate structured session cards and archive to Notion |
| `summary` | Strategic synthesis of current session |
| `status` | Fast tactical checkpoint of current session |
| `dev` | Claude code engine for all yOS dev projects |

## Usage

Skills are read by Manus at task start when relevant. Each skill contains a `SKILL.md` with instructions, templates, and examples.

## Source of Truth

This repository is the canonical source for all yOS Manus skills.
