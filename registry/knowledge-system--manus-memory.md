# Manus Memory — Y-OS Knowledge System

Manus Memory is the persistent memory layer of Y-OS. It uses Notion as the primary structured store and Mem0 for cross-session semantic retrieval.

## Components
- Notion: Sessions, Projects, Knowledge pages
- Mem0: Cross-session semantic memory
- Git: Canonical backup
