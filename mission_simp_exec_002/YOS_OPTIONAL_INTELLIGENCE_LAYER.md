## What Remains Useful

All components remain useful — they are not deleted. They are:

- Preserved in Git
- Accessible on demand
- Activated by explicit trigger
- Not loaded by default

---

## What Is Not Required for Daily Operation

Daily operation = one session, one context pack, one LLM call, one artifact.

None of the intelligence layer components are required for this cycle.

## When to Use

- Monthly strategic review
- Quarterly architecture audit
- Onboarding new context to a session
- Preparing a CEO briefing
- Evaluating a major architectural decision

## When NOT to Use

- Daily execution sessions
- Single-task LLM calls
- Quick artifact generation
- Any session under 30 minutes
