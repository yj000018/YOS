# Build Context — yOS

## What is yOS?

Y-OS is a cognitive operating system for a single founder (Yannick Jolliet). It is not a product — it is the infrastructure that enables all products.

## Core Components

- Manus (AI operator / copilot)
- Notion (thinking layer)
- GitHub (building layer)
- n8n (automation layer)
- NAS (persistence layer)

## Build Philosophy

- Notion for thinking, GitHub for building
- Single-founder system — no team management overhead
- Every project follows the same 19-page Notion structure
- Manus is the primary builder and executor
