# Y-DEV — Y-OS Development Protocol

Y-DEV is the meta-protocol governing all development activity within Y-OS. It defines the standards, phases, and quality gates that every component (Skill, Workflow, Agent, Service) must pass through before reaching Active status in Y-REG.

## Governs
- All Skills
- All Workflows
- All Agents
- All Services
