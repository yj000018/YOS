**Purpose:** Constitutional layer. Contains the foundational documents that define what YOS is, how it is organized, and what belongs where.

- YOS Constitution
- Module Registry
- Repo Map

**What does NOT belong here:**

- Knowledge content
- Execution logs
- Agent definitions
