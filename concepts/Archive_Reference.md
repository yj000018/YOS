# Archive Reference

**Type:** Concept  
**Domain:** Memory  
**Status:** CANONICAL  
**Constitutional Grounding:** Article II

---

## Definition

An Archive Reference is a pointer in the Y-OS memory system to a canonicalized artifact that has been archived but is no longer in active context. Archive References enable the Living Memory Pipeline to maintain a complete organizational memory without loading all historical artifacts into active context. They are the mechanism by which Y-OS achieves infinite memory with finite context.
