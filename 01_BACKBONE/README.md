# 01_BACKBONE — yOS Core Infrastructure

**Purpose:** All backbone modules that power the yOS system. These are infrastructure-level components, not projects or applications.

**What belongs here:** MPM (Mega Prompt Manager), KAP (Knowledge Absorption Pipeline), GOVERNANCE, ROUTING, MEMORY, BUS, SECURITY, and other core infrastructure modules.

**What does NOT belong here:** Applications, knowledge content, agent skill definitions, automations, or project-specific work.

**Key rule:** KAP is a backbone module. It belongs here, not in any projects folder.
