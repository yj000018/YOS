## Non-negotiable rules

- No repository is excluded merely because its name appears unrelated.
- No repository is deleted, archived, merged, or rewritten without explicit human approval.
- No newer term automatically erases an older concept.
- No old implementation claim is treated as current without evidence.
- No summary may replace a high-value original artifact.
- Consolidation must increase density, accuracy, traceability, and reuse value.
- Historical dead ends remain recorded when their reasoning or code may be reusable.
- Timeline, chronology, and supersession are first-class data.
- Existing branches, PRs, issues, commits, tags, workflows, docs, code, schemas, prompts, and generated artifacts are all evidence.
- Git is durable authority; Obsidian is the preferred human interface; Notion remains transitional/operational pending evidence, not an automatically equal source of truth.
