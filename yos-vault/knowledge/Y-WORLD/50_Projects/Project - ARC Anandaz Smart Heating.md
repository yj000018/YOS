# Project — ARC Anandaz Smart Heating

Implement a smart, predictive heating system for the Swiss chalet [[ARC Anandaz]] to balance comfort and energy efficiency during winter.

## Objectives

- [ ] Install Shelly TRVs (Thermostatic Radiator Valves) on all radiators
- [ ] Configure Home Assistant with custom schedules based on occupancy and Swiss weather forecasts
- [ ] Setup local voice control using ESP32-S3 hardware running local pipelines [[Anandaz Philosophy]]
- [ ] Pipe energy logs to [[ARC Anandaz Dashboard]]

## Linked Systems

- **Hardware**: Shelly TRVs + ESP32-S3 local voice nodes
- **Monitoring**: [[ARC Anandaz Dashboard]]
- **Philosophy**: [[Anandaz Philosophy]] (Cognitive silence — no noisy screens, physical tactile controls only)\n
