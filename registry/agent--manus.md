# Manus — Primary Y-OS Agent

Manus is the central execution agent of Y-OS. It can read and write to all connected systems, execute Skills, trigger Workflows, and interact with external APIs.

## Capabilities
- Skill execution
- Workflow orchestration
- Notion/Supabase/GitHub integration
- Web browsing and extraction
